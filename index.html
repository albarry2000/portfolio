Mes Projets

Ibentau chatbot

Ibentau est un chatbot qui peut être intégré au site web d'une conférence technologique. Il est capable de répondre aux questions des participants telles que l'horaire, l'hébergement ou la manière de soumettre un article. Il peut être entraîné par l'IA et se souvenir du contexte et des messages précédents. En outre, il peut également être géré par un administrateur, qui peut définir l'événement et le panel de réponses. Enfin, des outils tels que la reconnaissance des entités nommées peuvent être utilisés pour améliorer l'efficacité du chatbot.

  • NodeJS
  • OpenAI API
  • JavaScript

Projet d'Architecture logiciel

Il s'agit d'un projet pour le cours Web et Architecture à l'ESIR. L'objectif est de créer une application web avec une architecture microservice. L'application est un simple panneau d'administration pour une société qui gère des associations. L'application est composée d'un front-end, d'une API, d'un microservice de messagerie, d'une base de données, d'un broker de messages, d'un reverse proxy, d'un testeur de charge et d'outils de monitoring.

  • Docker
  • Quarkus
  • NestJS
  • PostgreSQL
  • RabbitMQ
